## Getting Started Back-end

Run the development server:

npm i 


Open [http://localhost:4000/usuarios/](http://localhost:4000/usuarios/) with your browser to see the result = API ok.


## Libs

- [Node Documentation](https://node.com.br) - Principal ferramenta.

## Comandos Utilizados

* npm init -y
* npm i express
* node --watch server.js
* npm i prisma --save-dev
* npx prisma init
* npx prisma db push
* npm install @prisma/client
* npx prisma studio
